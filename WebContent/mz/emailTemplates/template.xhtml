Vous pouvez utiliser comme champs dynamique :
  • ${'#{'}envoiDevisManager.formations${'}'} : Le libellé des formations concernées
  • ${'#{'}envoiDevisManager.participants${'}'} : Le nombre de particpants pour les formations concernées
  • ${'#{'}envoiDevisManager.lieu${'}'} : Le lieu des formations concernées
  • ${'#{'}envoiDevisManager.duree${'}'} : Les durées des formations (en jours et heures)
  • ${'#{'}envoiDevisManager.dureeSouhaitee${'}'} : Les durées souhaitées des formations (en jours et heures) (pour les intra)
  • ${'#{'}envoiDevisManager.date${'}'} : Les dates des sessions proposées
  • ${'#{'}envoiDevisManager.dateSouhaitee${'}'} : Les dates souhaitées par le prospect
  • ${'#{'}prospect.adresse${'}'} : L'adresse du prospect
  • ${'#{'}prospect.pays${'}'} : Pays du prospect
  • ${'#{'}prospect.cp${'}'} : Code postal du prospect
